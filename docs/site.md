# 文档站结构

文档站使用仓库内 Markdown 作为内容源，栏目如下：

- 首页：定位、快速验证命令、参考接入工程、报告输出、架构图。
- 快速开始：模拟器、图像输入、参考场景、报告生成。
- 架构：事件、任务、技能、机器人描述、适配器、运行时。
- 技能开发：`skill.yaml`、模板生成、校验、安装、运行。
- 适配器开发：串口、HTTP、MQTT、ROS 2、模板生成、契约测试。
- 场景运行：任务、机器人描述、技能包、适配器和期望结果。
- 模型提供方：规则解析、OpenAI 兼容接口、文本和图像任务解析。
- 可观测性：审计日志、任务回放、失败恢复、报告输出。
- 公开接口契约：事件、任务、技能、适配器、执行结果和声明文件。
- 贡献指南：技能包、适配器、示例和文档贡献流程。
