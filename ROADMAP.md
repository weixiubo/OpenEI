# OpenEI 长期开源路线

OpenEI 的长期方向是机器人版 Agent 运行时：让开发者用低成本真实机器人快速验证具身任务、技能编排和硬件适配。

对外表达保持“原型但有野心”：不宣传“首个”，重点强调低成本、五分钟可运行、技能可扩展、硬件可适配。

## 一个月内

- 完成 `openei` 框架骨架。
- 提供无硬件五分钟模拟器。
- 把当前动作元数据迁移成默认技能包。
- 提供串口适配器包装。
- 补齐架构、快速开始、技能开发、适配器开发文档。
- 录制一个模拟器演示和一个真实机器人执行视频。

## 三个月内

- 增加视觉或图像输入适配器样例。
- 增加多模型推理接口，支持替换不同模型服务。
- 提供更多机器人适配器模板。
- 补充贡献指南和插件式目录结构。
- 增加任务执行日志导出和基础审计能力。

## 六个月内

- 形成技能包生态雏形。
- 支持第三方技能目录。
- 支持硬件适配器模板生成。
- 增加任务执行回放、失败分析和安全约束策略。
- 形成一套面向开发者的教程、短文、动图和真实机器人演示材料。

## 传播节奏

1. 发布“机器人版 OpenClaw”概念短文和动图。
2. 发布五分钟模拟器教程。
3. 发布真实机器人执行视频。
4. 邀请社区贡献技能和硬件适配器。
